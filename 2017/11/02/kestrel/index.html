kestrel

Kestrel is a cross-platform web server for ASP.NET Core based on libuv, a cross-platform asynchronous I/O library.

Microsoftdocs.microsoft.com/en-us/aspnet/core/fundamentals/servers/kestrel?tabs=aspnetcore2x

通过微软的定义：可知Kestrel 是一个跨平台的，异步IO的library。
Kestrel支持：Https, websocket, Unix Socket

Kestrel工作方式：

  1. without reverse proxy
    Internet <->(http) ASP.NET Core application ( Kestrel <-> (httpContext) code )

  2. with reverse proxy:
    Internet <-> (http) Reverse proxy (IIS/windows, Nginx/linux) <->(http) ASP.NET CORE application ( Kestrel <-> (httpContext) code )

Tip：
反向代理：安全（多一层代理），load balance （平衡webserver负载），静态资源的缓存等等；另外，由于代理转发，故可以让多个web应用共享同一port和IP，

Kestrel在ASP.NET CORE中的应用：

IWebHostBuilder 扩展方法：UseKestrel()
同时可以设置限制的配置选项：

  1. MaximunClientConnection 最多的客户端连接数 default is null
  2. MaximunRequestBodySize 最大的请求Body大小 default is 30000000bytes
  3. MinimunRequestBodyDataRate Kestrel 检查传输速度，如果传输速度太小 close the connection.
